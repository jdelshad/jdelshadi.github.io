UCSF-ILLIOS CURRICULUM VISUALIZER

Click to Demo

A project created with JavaScript libraries (d3.js , Ember.js , node.js) with the aim of creating an interactive visualization of complex medical school curriculum for integration into Ilios Curriculum Manager.

On this project we collaborated with team of developers at UCSF School of Medicine. Project will be integrated in 15+ Health Professional school programs. Presentation to Client.

Project Highlights:

  • The Illios Visualizer Project replaces the need to static handmade visualization.
  • The project directly integrates with clients exsiting system.
  • This visualizer is scalable growing easily with the clients system.
  • Using the Sunburst and d3.js creates an interactive and intuitive model for users.

AeroEarth Weather

Click to Demo

The Aeroeath project, forked and extended functionality of an open source project to visualize and compare large sets of NOAA (Operational Model Archive and Distribution System) weather data. A student project sponsored by the Aerospace Corporation. Created on JavaScript code base running on Amazon EC2 Instance, this tool currently compares two days of weather data automatically highlighting diference in each days surface wind data.

ANTTRAILS

Click to Demo

A simple and interactive application for visually displaying information from the boothing locations around campus. Reserve your spot, see a map view of occupied spots , edit your spot information at any time with a password and all information is cleared every 24 hours.

This project utlized Google App Engine, Google Maps API to create and interactive tool for student vending at UC Irvine. AntTrails attempts to connect students to students to student vendors, increasing sale and forming a community.
