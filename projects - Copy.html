The Illios Curriculum Visualizer The Illios Curriculum Visualizer was created with JavaScript libraries (d3.js , Ember.js , node.js) with the aim of creating an interactive visualization of complex medical school curriculum for integration into Ilios Curriculum Manager.

On this project we collaborated with team of developers at UCSF School of Medicine. Project will be integrated in 15+ Health Professional school programs.

Project Highlights:

  • The Illios Visualizer Project d3.js replaces the need to static handmade visualization.
  • The project directly integrates with clients exsiting system.
  • This visualizer is scalable growing easily with the clients system.
  • Using the Sunburst and d3.js creates an interactive and intuitive model for users.
